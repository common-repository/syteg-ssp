== Description ==

= Live Chat features =

Support your customers via Syteg SSP Live Chat. It is free and user-friendly. Build relationship with your customers through real-time conversation.
Livechat plugin from Syteg SSP has the following features:

* Toll free. No cost livechat that can be easily installed
* Fully-adaptive window design
* Offline chat messages to email (if chat operator is not online)
* Change the colors of your message-box theme
* Choose where you want to position the chat box on your website
* Сustomize the message box lookout with own avatar image
* No coding needed. Fully adaptive
* User-friendly and responsive design
* File upload and download for documents and photos
* Contact your visitors via email when they exit the chat
* Receive feedback from your customers
* Receive chat transcript on your email
* Responsive design that adapts to smartphones of any size

== Frequently Asked Questions ==
= Installation =
* Create an account on [syteg.com](https://syteg.com)
* Go to [Syteg SSP panel](https://call-center.syteg.com/new/)
* Get External Chat line number that you are going to use  (You can find your chat line number in Services & Employees Tab).
* Set up the service in plugin
* Sign in to Syteg SSP panel, and start using free Live chat.
= Is it free? =
Yes. You are welcome to use Livechat plugin from Syteg SSP free of charge.
= Do I need to create a syteg account to use the plugin? =
Yes. You can create it [here](https://syteg.com/register).
